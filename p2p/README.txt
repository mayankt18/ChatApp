This is the peer-to-peer version of the chat app made using the socket library in python.

REQUIREMENTS :
  - python 3 
  - tqdm (after installing python3 just type "pip install tqdm" in terminal or command prompt)
  
FEATURES :
  - you can send text messages using the chat app
  - you can send files using this chat app
  
HOW TO USE :
  - run the program using the terminal, just type "python3 chat.py" in terminal to run the app.
  
KEY CONCEPT USED :
  - socket programming
  - threading 
  - dividing the files in packets to send and receive files.
